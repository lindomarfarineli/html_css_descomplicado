HTML - Estrutura Básica

Para iniciar um arquivo html, é necessário que primeiramente, o arquivo seja nomeado com a extensão .html, ou seja, o arquivo receberá o nome algumNome.html. Já o corpo do arquivo deverá apresentar a sua primeira tag, que será <!DOCTYPE>.

Esta tag pode ser escrita da forma que foi demonstrado, ou seja em maiúsculo e sem fechamento, ou ainda em minúsculo ou só com a primeira letra maiúscula, o resultado é o mesmo.

É esta tag que, aliada a extensão .html do arquivo indicará para o navegador o uso do HTML5 e portanto, como abrir e executar este tipo de arquivo.

Um arquivo .html é composto por tags. Uma tag possui a seguinte estrutura:

<abertura> conteúdo </fechamento >

Para fazer a tag de abertura, há o sinal de menor-que(<), o nome da tag e depois o sinal de maior-que(>).
Para fazer a tag de fechamento, há o sinal de menor-que(<), uma barra(/), em seguida o nome da tag e depois o sinal de maior-que(>).
Entre a tag de abertura e de fechamento fica o conteúdo que será exibido. Nem todas as tags possuem conteúdo, e algumas das tags que não possuem conteúdo também não possuem tag de fechamento. Vide por exemplo a tag de linha horizontal <hr/>.

Existem ainda tags que apesar de não possuirem conteúdo, possuem parâmetros, como por exemplo a tag image <img>.

Logo abaixo da tag <!DOCTYPE> deve-se colocar a tag <html>, esta tag possui fechamento mas não há nada além disso. Todo o arquivo html, isto é, o conteúdo, ficará dentro desta tag.

O espaço da tag <html> é divido pelo menos em duas partes. As tags <head> e <body>.

A tag <head> possui conteúdo que não necessáriamente será mostrado na página. Um desses conteúdos é a tag <title>, responsável pelo título do site que aparece na aba do navegador.
Outra configuração que deve ser colocada é a tag <meta charset="UTF-8">. Esta configuração deve ser utilizada para que sejam reconhecidos acentos, cedilha e etc.

Dentro da tag <body> deve ficar todo o site propriamente dito

Veja abaixo um exemplo básico de um arquivo HTML, note onde ocorrem as aberturas e fechamentos de tags:

<!DOCTYPE>
<html>
   <head>
      <title>título do seu site</title>
      <meta charset="UTF-8"
    </head>
    <body>
      todas as outras tags e divisões ficarão aqui
    </body>
</html>

Outras Informações Que Podem Ser Relevantes.

Esta é a estrutura básica do HTML. Outras informações poderão ser acrescentadas a tag head.

A parte do site chamada comumente de body pode conter apenas a tag a tag <body> ou pode existir ali 3 divisões para conteúdos feitas com as seguintes tags:

  • <header> -- para fazer um cabeçalho
  • <body> --o corpo central do site
  • <footer> --o chamado rodapé do site
