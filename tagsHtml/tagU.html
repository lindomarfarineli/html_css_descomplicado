Sublinhando Uma Palavra Ou Frase - Tag <u>

Para sublinhar um texto usa-se a tag <u>. Veja abaixo a forma de utilização:

<p>Um parágrafo qualquer que conta uma  <u>história</u>  de algo.</p>

Note a palavra 'história' dentro de uma tag <u>. Perceba que esta tag está dentro de um parágrafo. Veja abaixo como ficaria o exemplo acima:

Um parágrafo qualquer que conta uma história de algo.

Outras Informações Que Podem Ser Relevantes.

A tag <u> não possui semântica, é meramente decorativa.
