Links - Utilizando A Tag <a >

Para criar links usa-se a tag <a>, e dentro dela há o parâmetro "href". Este é o parâmetro responsável pela navegação.

Os links podem ser externos ou internos.
Um link externo direciona o usuário para uma página externa, fora do seu site.
Um link interno direciona o usuário para uma página do próprio site ou para uma seção da própria página.

Veja abaixo um exemplo de navegação externa por meio de url, onde é fornecida a url do site do google:

<a href="https://www.google.com.br" target="_blank" > Vai para a página do google </a>

Note ainda o parâmetro target="_blank", que serve para que a página seja aberta em uma nova aba.

Observe abaixo a implementação do exemplo:

vai para a página do google

Embora o navegador tenha tamanhos determinados e tipos já determinados para esta tag, como de hábito, tudo pode ser customizado de acordo com a criatividade de quem está estruturando do site. É possível alterar tamanhos de fontes, as próprias fontes e até mesmo suas cores.

Outras Informações Que Podem Ser Relevantes.

O nome da tag <a> deriva de anchor, em português, "âncora".

O parâmetro 'target' é opcional. Caso esse parâmetro não seja especificado, a página será aberta da maneira padrão, isto é, na mesma aba que o usuário está.

Para páginas de seu próprio site, um desenvolvedor provalmente criaria links em algum lugar na página para que o usuário possa retornar e continuar navegando no seu site, mas se o link redireciona para site externo, como você deve ter notado caso tenha clicado no link do exemplo, significa que o usuário deixa totalmente o site em que vinha navegando.

Por esse motivo, a opção "_blank" geralmente é utilizada para navegação externa, quando o criador do site precisa direcionar o usuário para sites de terceiros, mas quer que o seu próprio site continue aberto, para que o usuário tenha oportunidade de voltar facilmente, caso queira.
