    • Quebrando Uma Linha Com a Tag <Br/>

    Pule uma linha, deixe tudo mais legível!

    Várias tags quebram linha automaticamente, como por exemplo a tag <p>, porém, caso se utilize a tag <a> e a tag <img> juntas, elas aparecerão na mesma linha, pois não possuem quebra de linha automática. Se esta não é a intensão do desenvolvedor, é sua obrigação providenciar manualmente a quebra de linha. Isto é feito utizando a tag <br/>.

    Veja o exemplo abaixo:

    <br/>

    Note que esta tag não possui parâmetros ou propriedades e não possui tag de fechamento já que o fechamento é feito do lado direito da tag.


    Outras Informações Que Podem Ser Relevantes.

    Note que esta tag não possui tag de fechamento. Caso o usuário faça algo como <br> </br> é considerado um erro e além disso o navegador provavelmente pulará 2 linhas. Todavia caso o desenvolvedor apenas utilize a tag sem a barra, como em <br>, é amplamente aceito e não causa erro nos navegadores.

    Outro detalhe importante é que este elemento deve ser utilizado em casos onde a quebra de linha de fato se faça necessária, como o caso ilustrado na explicação principal. Caso o objetivo seja escrever um texto, prefira a tag <p> ou na necessidade de simplesmente aumentar espaço superior ou inferior, prefira sempre a propriedade margin do css.

    É mais adequado e muito mais elegante.
