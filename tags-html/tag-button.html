    • Criando Um Botão Com a Tag <button>

    Crie botões para facilitar a navegação...e para o que mais precisar!

    As vezes é super necessário um ter botão na sua página, seja para enviar um formulário ou mesmo para fazer uma determinada confirmação.

    Para criar um botão basta utilizar a tag <button> como mostrado no exemplo:

    <button>Clique aqui</button>

    É óbvio que da forma como está demonstrado este botão não terá efeito algum. Em breve haverá mais explicações.
