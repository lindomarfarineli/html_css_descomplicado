Formatando um texto

Nesta seção serão abordadas várias maneiras de realizar formatações à partir de tags html:

  • Negrito
  • Sublinhado

Colocando Um Texto Ou Uma Palavra Em Negrito

Para colocar um texto em negrito é possível utilizar a tag <strong>. Veja abaixo a forma de utilização:

<p>Um parágrafo qualquer que conta uma  <strong> história </strong>  de algo.</p>

Note a palavra 'história' dentro de uma tag <strong>. Perceba que esta tag está dentro de um parágrafo. Veja abaixo como ficaria o exemplo acima:

Um parágrafo qualquer que conta uma história de algo.

Para mais informações, veja informações relevantes sobre negrito.


Sublinhando Um Texto Ou Uma Palavra

Para sublinhar um texto usa-se a tag <u>. Veja abaixo a forma de utilização:

<p>Um parágrafo qualquer que conta uma  <u>história</u>  de algo.</p>

Note a palavra 'história' dentro de uma tag <u>. Perceba que esta tag está dentro de um parágrafo. Veja abaixo como ficaria o exemplo acima:

Um parágrafo qualquer que conta uma história de algo.

Para mais informações, veja informações relevantes sobre sublinhado.


Outras Informações Que Podem Ser Relevantes.

Negrito

A tag <strong> funciona bem com o negrito, porém, trás em si propriedades semânticas utilizadas nos motores de busca. Assim, está deve ser reservada realmente para quando é necessário criar um destaque no texto.
Caso o objetivo seja simplesmente decorativo, como chamar a atenção para um vocábulo, mas que não seja algo que mereça uma atenção maior, existem outros meios de utilizar o negrito, como a tag decorativa <b>.

Sublinhado

A tag <u> não possui semântica, é meramente decorativa.
