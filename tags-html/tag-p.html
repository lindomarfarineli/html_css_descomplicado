Parágrafos em Html - Tag <p>

A tag <p> é utilizada para fazer parágrafos. Assim, já possui configurados por padrão espaçamentos e quebras de linha apropriadas para parágrafos.
Veja abaixo a forma de utilização:

<p>Um parágrafo qualquer que conta uma história de algo.</p>

Veja abaixo como ficaria o exemplo acima. Os espaçamantos são resultado da tag p. Veja abaixo:

Um parágrafo qualquer que conta uma história de algo.

Outras Informações Que Podem Ser Relevantes.

É possível colocar outras tags dentro da tag <p>, tais como:

  • negrito,
  • sublinhado,
  • riscado,
  • itálico,
  • uma lista como esta, etc.

A importância da tag <p> não é meramente visual. Esta tag tem importância semântica também. O que é chamado de SEO(otimização para mecanismos de pesquisa). Ou seja, seu texto será classificado como um parágrafo.

Caso queira chamar atenção para seu parágrafo, coloque antes um título sugestivo utilizando a tag <h1> ou outra tag <h> que faça sentido para seu texto, mas procure utilizar as palavras mais importantes do(s) parágrafo(s).

Separar um conteúdo em parágrafos torna um texto mais fácil de ler para todas as pessoas. Leitores de tela e outras tecnologias possuem atalhos que permitem a navegação entre parágrafos produzindo uma leitura rápida do texto.

Utilizar elementos <p> vazios simplesmente para adicionar linhas em branco é uma prática condenável para quem depende dessas tecnologias. O leitor anunciará a existência de um parágrafo mas não haverá nenhuma leitura, já que não há conteúdo. Isso atrapalha e pode tornar o texto confuso para quem depende dessas tecnologias.
