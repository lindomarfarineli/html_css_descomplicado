Criando Listas Estruturadas e Não Estruturadas

Para criar listas, podem ser utilizados dois tipos de tags diferentes, que produzirão resultados diferentes. São elas, a tags:

  • <ul>
  • <ol>

Ambas servem para fazer listas, mas primeiramente será abordado o contexto da tag <ul>.

Uma lista, qualquer que seja ela, é composta de itens, então, os itens são colocados dentro da lista por meio da tag <li>, derivada de List iten.

A tag <ul> serve para produzir uma lista desordenada, disto deriva seu nome de unordered list, ou seja, é uma lista onde a ordem não importa, no próximo exemplo, não haveria problema algum em inverter a ordem das cores. Veja abaixo:

<ul>
   <li>azul</li>
   <li>vermelho</li>
   <li>amarelo</li>
</ul>

O resultado exato do código acima está executado abaixo:

  • azul
  • vermelho
  • amarelo

Diferentemente, as lista feitas pelas tags <ol>, de ordered list são produzidas da mesma forma, mas levando em conta que ordem importa:

<ol>
   <li>acordar</li>
   <li>tirar a roupa</li>
   <li>tomar banho</li>
   <li>secar-se</li>
   <li>colocar roupa</li>
   <li>ir para o trabalho</li>
</ol>

A diferença, é que nas listas ordenadas haverá um elemento de ordem, como no exemplo, os números a esquerda, veja o resultado abaixo:

  1. acordar
  2. tirar a roupa
  3. tomar banho
  4. secar-se
  5. colocar roupa
  6. ir para o trabalho

Se a ordem for trocada, o indivíduo corre o risco de chegar ao trabalho completamente molhado ou até mesmo pelado.
