Oquê eu vou encontrar aqui?

Este site foi criado por um estudante de HTML & CSS que nunca lembrava que tag faz o quê...Assim, no índice de casos logo abaixo, você não vai encontrar o nome da tag (já que se você sabe qual tag é você usa né...). Ao invés disso, vacê vai encontrar o caso de uso, ou seja a ação que você quer fazer.

Isto vai te levar direto para uma página que explica como é possível fazer oquê você deseja. Mas, entendemos que talvez você saiba qual tag ou comando você quer, mas talvez falte um detalhe ou queira tirar uma dúvida...ok.

Neste caso você procure o índice de Tags, talvez você encontre o que precisa.

Por fim, muito obrigado por utilizar o site, e contribuições são sempre bem vindas, por hora, como este site ainda se encontra em desenvolvimento, utilize o email lindomarfarineli@hotmail.com, futuramente disponibilizaremos outros meios de contato!
