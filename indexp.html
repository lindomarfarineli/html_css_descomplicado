    • Sobre HTML, CSS e suas tags.

    O objetivo deste site, é trazer para o estudante de HTML5 e CSS dicas e informações que possam facilitar seu aprendizado.

    Sabemos que decorar tudo é impossível, a não ser que você seja um profissional que utiliza tudo constantemente, dificilmente conseguirá decorar todas as tags, parâmetros e atributos.

    Embora decorar seja possível não é nem de longe necessário, e na verdade, a maioria das pessoas não decora. É mais importante conhecer e entender o funcionamento. Quando for necessário, caso você não se lembre, basta uma pesquisa rápida para 'lembrar' o que é necessário, inclusive a sintaxe, etc.

    Assim, este site utiliza linguagem simples, que um estudante pode entender facilmente e utiliza apenas os recursos dos quais trata no seu assunto principal, ou seja, este site foi produzido utilizando apenas HTML e CSS, nada mais, nenhum 'framework', nenhum facilitador como bootstrap, nada! Apenas o bom e velho HTML e CSS.

    Ao invés de encontrar apenas o nome da tag e uma explicação genérica, aqui você também vai encontrar um índice de casos de uso para HTML e CSS!

    Assim, se você não sabe qual a tag, mas sabe o que precisa ser feito, vá em frente!

    Isto vai te levar direto para uma página que explica como é possível fazer o que você deseja.

    Mas, entendemos que talvez você saiba qual tag ou comando você precisa usar, talvez falte um detalhe ou queira tirar uma dúvida...ok. Neste caso você procure o índice de Tags. Talvez você encontre o que precisa.
