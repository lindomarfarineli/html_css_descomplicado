Deixando Uma Palavra Ou Frase Em Negrito

Para colocar um texto em negrito é possível utilizar a tag <strong>. Veja abaixo a forma de utilização:

<p>Um parágrafo qualquer que conta uma  <strong> história </strong>   de algo.</p>

Note a palavra 'história' dentro de uma tag <strong>. Perceba que esta tag está dentro de um parágrafo. Veja abaixo como ficaria o exemplo acima:

Um parágrafo qualquer que conta uma história de algo.

Outras Informações Que Podem Ser Relevantes.

A tag <strong> funciona bem com o negrito, porém, trás em si propriedades semânticas utilizadas nos motores de busca. Assim, está deve ser reservada realmente para quando é necessário criar um destaque no texto. Caso o objetivo seja simplesmente chamar a atenção para um vocábulo, mas não seja algo que mereça uma atenção maior, existe outros meios de utilizar o negrito, como a tag...
